什么是js curry(柯里化)：
Currying 只传递给函数一部分参数来调用它，让它返回一个函数去处理剩下的参数。
原理：
function Currying(fn, ...args1) {
  return function(...args2) {
    return fn(...args1, ...args2)
  }
}

const add = (...args) => {
  let sum = 0
  for (let a of args) {
    sum += a
  }
  return sum
}

const increament = Currying(add, 1)
increament(2) === 3

什么是闭包：
一般来说函数外部无法获取函数内部的局部变量，
但是可以闭包的方法结局。
本质就是函数内部和函数外部链接起来的一座桥梁
因此可以芭比宝理解成为定义在一个函数内部的函数。
作用：
闭包是一种保护私有变量的机制，在函数执行时形成私有的作用域，保护里面的私有变量不受外界干扰。
直观的说就是形成一个不销毁的栈环境。
