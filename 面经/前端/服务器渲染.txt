回流=> 得到dom的位置信息
重绘=> 得到dom的像素组成

按照图片里我理解的重绘和回流都是渲染过程中的一步
比如你在页面画一个三角形
第三步得到几何信息是知道其在页面中所在的位置，这个三角形是由多少个像素组成的
第四步是得到每一个像素在页面上的坐标（实际像素）
所以可以看到回流一定引发重绘，但重绘不一定引起回流

处理 HTML 并构建 DOM 树。
处理 CSS 构建 CSSOM 树。
将 DOM 与 CSSOM 合并成一个渲染树。
根据渲染树来布局，计算每个节点的位置。
调用 GPU 绘制，合成图层，显示在屏幕上。

1. 生成渲染树 tree
2. 回流 position
3. 重绘 pixel
4. 渲染页面 render
