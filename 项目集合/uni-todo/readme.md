# 这是一个uniapp demo
===
_这个是webstrom中开发uni-app的链接_
[点击跳转](https://ask.dcloud.net.cn/article/36307)<br>
### cli工程
```node
vue create -p dcloudio/uni-preset-vue my-project
npm run dev:%PLATFORM%
npm run build:%PLATFORM%
```
| 值 | 平台 |
| :---: | :---: |
| h5| H5|
|mp-alipay	|支付宝小程序
|mp-baidu	|百度小程序
|mp-weixin	|微信小程序
|mp-toutiao	|头条小程序
|mp-qq	|qq 小程序
### HBuilderX工程
```node
npm init -y
npm i @types/uni-app @types/html5plus -D
```
